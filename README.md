# jeremy.ng

This is my personal website. It was built using Next.js, React, and TypeScript.

## Installation

```bash
git clone https://github.com/jeremy-code/jeremy.ng
cd jeremy.ng
yarn
```

## Usage

```bash
# Run the development server locally
yarn dev

# Lint and format code
yarn lint
yarn format

# Build
yarn build
```
